## O que é uma Rede de Computadores?

Uma rede de computadores é um conjunto de dispositivos interconectados que compartilham informações e recursos. Pense na internet: você acessa sites, faz chamadas de vídeo, joga online e baixa arquivos. Tudo isso acontece graças às redes. Mas não só a internet – redes também conectam computadores dentro de empresas, universidades e até mesmo na sua casa.

### Exemplos de Redes no Cotidiano:

- **A rede Wi-Fi da sua casa** que conecta celulares, notebooks e Smart TVs.
- **A rede de computadores de um banco**, que interliga terminais de autoatendimento e servidores.
- **Sistemas de comunicação como WhatsApp e Zoom**, que usam redes de servidores distribuídos.
- **Sistemas de pagamento**, como cartões de crédito conectados a redes bancárias.

## Tipos de Redes

As redes podem ser classificadas de acordo com sua abrangência e propósito:

- **LAN (Local Area Network)**: Rede local, geralmente dentro de um ambiente fechado, como uma casa, escritório ou laboratório de informática.
- **MAN (Metropolitan Area Network)**: Rede metropolitana, conectando vários pontos dentro de uma cidade, como redes de universidades ou redes de transporte público.
- **WAN (Wide Area Network)**: Rede de longa distância, como a internet, que interliga computadores ao redor do mundo.
- **WLAN (Wireless Local Area Network)**: Versão sem fio de uma LAN, como as redes Wi-Fi.
- **VPN (Virtual Private Network)**: Rede privada virtual, utilizada para conectar usuários de forma segura a redes corporativas.

## Componentes Básicos de uma Rede

Para que uma rede funcione, são necessários alguns elementos essenciais:

- **Dispositivos de Rede:** Roteadores, switches, hubs, servidores e computadores.
- **Meios de Transmissão:** Cabos de rede (Ethernet), conexões Wi-Fi, fibra óptica.
- **Protocolos de Comunicação:** Conjuntos de regras que garantem a comunicação eficiente entre os dispositivos, como TCP/IP e HTTP.

### Como os Dispositivos se Comunicam?

Os dispositivos trocam informações usando protocolos de comunicação. O mais conhecido é o **Protocolo de Internet (IP)**, que permite que os dados sejam enviados de um ponto ao outro de forma organizada e confiável. Além dele, outros protocolos importantes incluem:

- **TCP (Transmission Control Protocol):** Garante a entrega confiável dos pacotes.
- **UDP (User Datagram Protocol):** Mais rápido, mas sem garantia de entrega.
- **HTTP/HTTPS:** Protocolos usados para navegação na web.

### Experimento Prático

Para tornar esse conceito mais claro, vamos realizar um teste simples usando o comando `ping`.

- Abra o prompt de comando no seu computador.
- Digite `ping google.com` e observe a resposta.
- Esse comando verifica se o computador consegue se comunicar com o servidor do Google e mede o tempo de resposta.

### Como os Dados Trafegam na Internet?

Quando você acessa um site, muitas coisas acontecem nos bastidores:

1. Seu computador envia uma solicitação para um **servidor DNS** para traduzir o nome do site em um endereço IP.
2. O navegador estabelece uma conexão com o servidor do site via **HTTP ou HTTPS**.
3. Os dados do site são transferidos para seu computador em pacotes, passando por **roteadores e switches**.
4. O navegador reconstrói os pacotes e exibe a página.
